# Simple Alarm System For Detecting Intruder
Tutorial pembuatan alarm system sederhana menggunakan arduino uno.

## Alat dan Bahan
- Breadboard
- Arduino Uno
- Sensor Ultrasonik (HC-SR04)
- Buzzer
- Kabel Jumper
- LED
- Resistor

## Perakitan
Rakit Sesuai dengan desain fritzing dibawah.
